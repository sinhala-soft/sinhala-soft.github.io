Our Mission

The Sinhala Software Foundation is dedicated to developing and maintaining open source tools that make it easier to work with the Sinhala language in software development. We believe that language should never be a barrier to technology, and we're committed to building solutions that enable developers to create applications that support Sinhala seamlessly.
